# Secret Diary
## Web App designed for storing people daily memories inform of a digital diary!


#### Tech Stack Used :
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery, AJAX
- MySQL
- PHP

#### Functionality :
- Basic LogIn SignUp Functionality.
- Auto save text without refresh using AJAX request.
- Userauthentication and validation
